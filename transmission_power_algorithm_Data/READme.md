
# Transmission Power Algorithm Data — README

This folder contains data, scripts, and notebooks used to analyze transmission power and related algorithmic behavior for our experiments. The README provides quick instructions to regenerate plots and run any notebooks contained here.

## Contents

This folder currently contains:

- `TX_ML_Algo.ipynb` — Jupyter notebook that trains/evaluates an adaptive TX power classifier and generates analysis plots
- `TX_Power_Distances_Dataset.csv` — dataset used by the notebook (RSSI, throughput, power, distances, etc.)

Any plots generated by the notebook are saved alongside the notebook (or shown inline). Known output image files produced by `TX_ML_Algo.ipynb`:

- `feature_importance.png` — bar plot of top features used by the classifier
- `optimization_analysis.png` — plot summarizing optimization / parameter sweep results


## Prerequisites

Install a minimal set of Python packages commonly used by the notebooks:

```bash
python -m pip install pandas numpy matplotlib seaborn jupyter openpyxl
```

Additional packages may be required by specific notebooks — check the first code cells of any notebook for exact imports.

## Generating Plots

1. Change to this directory:

```bash
cd Transmission_power_algorithm_Data
```

2. Start Jupyter and open the notebook you want to run, or open the notebook in VS Code. Run all cells from top to bottom.

```bash
jupyter notebook
# then open the desired notebook and select "Run All"
```

3. Output images are saved by the notebook into the current directory (see the `plt.savefig(...)` calls). For this notebook the two known saved images are `feature_importance.png` and `optimization_analysis.png`. The notebook also displays an interactive/comprehensive figure with `plt.show()`.
